Five Flexbox Parent (container) Properties

(shorthand properties not included)
flex-direction
row (default), row-reverse, column, column-reverse
flex-wrap
nowrap (default), wrap, wrap-reverse
justify-content
think MAIN AXIS
flex-start (default) plus five other values
think (MS Word "justify" icon)
aligns items along the main axis
align-items
think CROSS AXIS
stretch (default) plus six other values
aligns items along the cross axis
align-content
aligns entire content within the flex container
avoid . . . confusing it is

Five Flexbox Child (items) Properties

(don't use within the container declartion block, but with the child items)

(shorthand properties not included)
order
dictates the order of the flex items (mostly obvious)
default is 0. Negative numbers are allowed
only works on flex items
flex-grow
The flex-grow property specifies how much the item will grow compared to others items inside that container.
default is 0 which means it will not grow at all.
"1" means it will grow 100%, "2" means it will grow 200%, etc.
courtesy https://www.geeksforgeeks.org/css-flex-grow-property/
flex-shrink
The flex-shrink property specifies how much the item will shrink compared to others items inside that container.
default is 1 which means it will shrink 100%
"1" means it will grow 100%, "2" means it will grow 200%, etc.
courtesy of https://www.geeksforgeeks.org/css-flex-shrink-property/
(just know what it is)
flex-basis
initial length of a flex item
default is auto which means . . .
"The length is equal to the length of the flexible item. If the item has no length specified, the length will be according to its content"
Excellent Example! https://www.w3schools.com/cssref/css3_pr_flex-basis.php
(just know what it is)
align-self
aligns item along the cross axis within the container
default is auto
"The value of (the auto) property is adjusted automatically according to the content or the context of the element"
(source for above info)
not useful for menus because "height: auto;" property/value combo make vertical alignment unimportant

Other Flexbox Properties

display
value is "flex" (this value turns flexbox on)
(not unique to Flexbox--but it is required)
default is the HTML specification for that element (block or inline)
numerous other values
flex-flow
shorthand combo for the parent properties, flex-direction & flex-wrap
default is flex-flow: row nowrap;
flex
shorthand combo for the three child properties: flex-grow, flex-shrink, and flex-basis
default is 0 1 auto;
